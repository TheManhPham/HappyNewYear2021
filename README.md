# Vim年賀状

これはVimで年賀状を作るためのリポジトリです

## Vimで年賀状を作るためにすること

1. ターミナルエミュレータのサイズを調整します 
    - e.g : iterm : `112*25`

2. `vim -N -u autoscript.vim` を実行します
