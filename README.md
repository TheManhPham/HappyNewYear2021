# Vim年賀状

これはVimで年賀状を作るためのリポジトリです


## Vimで年賀状を作るためのメモ

- itermのwindow size `112*25`
- 実行コマンド `vim HappyNewYear2021 -c "set nonumber | set showmode | set ft=HappyNewYear"`
